HELP: Quick System Info (GUI)

Displays a Quick System Info (QSI) report based on inxi and a few custom parameters for MX Linux.

The Quick System Info GUI application can display any log or report shown in the list (left pane) by clicking on an item.

Use the "Copy for forum" button at the bottom of the report display (right pane) to copy the report to the clipboard in a format suitable for directly pasting into a forum post, complete with code tags.
If a portion of text is seleted, only that portion is copied and encapsulated in code tags.

Use the "Save" button to save multiple selected reports into an archive file.

Use the "Save text" button to save the text from the report display to a text file.

Menus and Shortcut Keys

Right-clicking on a pane will trigger a pop-up menu, which can also be used to perform similar actions to the buttons.

PaneMenu ItemShortcutAction
ListSelect AllCtrl+A*Selects all reports to be saved to an archive file.
Revert SelectionCtrl+Shift+ASelects only the reports that were selected when the program started, unselecting the others.
SaveCtrl+SSame as the "Save" button.
TextCopy for forumCtrl+CSame as the "Copy for forum" button.
Plain text copyAlt+CCopies the text displayed as-is to the clipboard. If a portion of text is selected, only that portion is copied to the clipboard.
Save textCtrl+Shift+SSame as the "Save text" button.
FindCtrl+FSearch for text in the report display.
Find NextF3Find the next occurrence of the text from the previous search.

* If the report display pane is in focus, Ctrl+A selects all the text in the report display, instead of selecting items in the list.

Resizing the report list

To resize or collapse the list pane, place the cursor in the handle between the two panes and drag it to the preferred size. Double-clicking this handle will resize the list pane to best fit its contents.
