HELP: Quick System Info (gui)

Displays a Quick System Info (QSI) report based on inxi and a few custom parameters for MX Linux.

Also can display any log or report shown in the Left Pane by clicking on an item

Use the "Copy for Forum" button at the bottom of the report display to copy the report to the clipboard in a format suitable for directly pasting into a forum post, complete with code tags.

Use the "Save" button to save the current report into a file.

Use the "Save Selected" button to save multiple selected reports into an archive file.

Use the "Save" button to save the report to a text file

Shortcut keys: Alt+C plain text copy, CTRL+C Copy for forum

Development history: MX Linux
